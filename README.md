# AltBOM - AI 驱动的硬件替代料与降本雷达 🚀

AltBOM 是一款专为智能硬件研发与供应链采购打造的 **AI 辅助器件平替与降本分析工具**。它能够智能解析工程师上传的 BOM（物料清单）CSV 文件，并通过双通道混合检索算法（语义向量检索 + 精准硬件关键字过滤）快速在元器件库中匹配出引脚兼容（Pin-to-Pin）、性价比最高且现货充足的国产/替代元器件，出具直观的降本雷达分析清单。

---

## 🛠️ 技术架构 (Architecture)

系统采用了 **RAG (检索增强生成)** 与 **混合检索 (Hybrid Search)** 架构：

```
                    ┌──────────────────────────┐
                    │  上传 BOM 清单 (无价格)  │
                    └─────────────┬────────────┘
                                  │
                       [BOM Parser 表头对齐]
                                  │
                                  ▼
                    ┌──────────────────────────┐
                    │      AI 提取元器件型号    │
                    └──────┬─────────────┬─────┘
                           │             │
                    [语义向量通道]   [精准关键字通道]
                    (Cosine Sim)     (封装/电压过滤)
                           │             │
                           ▼             ▼
                    ┌──────────────────────────┐
                    │    混合得分融合 (RRF)    │
                    └─────────────┬────────────┘
                                  │
                       [立创/DigiKey 实时比价]
                                  │
                                  ▼
                    ┌──────────────────────────┐
                    │   降本雷达看板与引脚比对  │
                    └──────────────────────────┘
```

1. **BOM Parser (表头语义对齐)**：由于各个 CAD 软件导出的 CSV 表头极不一致，大模型会对 CSV 进行语义表头对齐（如 `位号` ➡️ `Designator`）。
2. **混合检索双通道**：
   * **向量检索通道 (Vector Search)**：将器件描述转化为特征向量（本系统核心基于 TF-IDF 向量空间模型实现），通过计算**余弦相似度 (Cosine Similarity)** 匹配功能近似的芯片。
   * **关键字检索通道 (Keyword Search)**：对芯片硬件硬性指标（如封装 `LQFP48`、`SOT-23`，电压 `3.3V`）进行过滤硬匹配，防范向量检索“因模糊语义推荐错引脚数”导致烧板的灾难。
3. **降本雷达可视化**：前端使用高质感的毛玻璃 (Glassmorphism) 暗黑设计，实时渲染混合得分，并提供一键穿透到**立创商城官方实时页面验证现货价格**的链接入口。

## 🚀 快速启动指南 (Quick Start)

项目被设计为**无缝双保险模式**：即便你不运行后端，前端网页也会自动降级使用内置离线高保真仿真数据包，保证你的黑客松路演 100% 不翻车！

### 1. 启动 Python 混合检索后端 (可选)
在终端执行以下命令，启动 API 网关（监听 `5000` 端口）：
```bash
python3 server.py
```

### 2. 启动前端展示
* **双击运行**：在文件浏览器中，直接双击 `index.html` 即可在任意浏览器中打开。
* **本地 Web 服务运行**：
  若你配置了 `bun` 环境，可以在项目根目录运行：
  ```bash
  bun x serve .
  ```
  或者用 Python 启动：
  ```bash
  python3 -m http.server 8000
  ```
  然后访问 `http://localhost:8000/index.html`。

### 3. 演示故事线 (Demo Storyboard)
1. 启动网页，将目录下的 `Smart_Watch_BOM_v1.0.csv` 拖入左侧拖拽区（或点击加载演示模板）。
2. 点击 **“AI 智能分析 & 降本优化”**。
3. 观察左侧酷炫的混合检索、向量匹配和表头对齐日志滚屏。
4. 右侧结果淡入，显示降本 **60.8%** 指标。点击器件旁的 **🌐 图标**，可直接跳转至立创商城官方页面实时查验真实现货价格，极具震撼力！
